To install the Miru Agent, please visit the [device provisioning](/docs/learn/devices/provision) section, which provides detailed instructions for installing and activating the agent via the following methods:

- [Dashboard](/docs/learn/devices/provision#dashboard) - a simple web interface for installing the agent on individual devices
- [API Keys](/docs/learn/devices/provision#api-keys) - a programmatic way to install the agent on many devices
- [Poor Connectivity](/docs/learn/devices/provision#poor-connectivity) - a method for installing the agent on devices with poor connectivity

## Upgrade

To upgrade a device's Miru Agent to a new version, simply install the agent.

**Dashboard**

To upgrade a device from the dashboard, follow the [dashboard provisioning](/docs/learn/devices/provision#dashboard) instructions as normal. The device will be automatically be reactivated and the agent upgraded to the latest version.

**API Keys**

To upgrade a device [via API Keys](/docs/learn/devices/provision#api-keys), rerun the provisioning process with the [`--allow-reactivation`](/docs/learn/devices/provision#param-allow-reactivation) flag. This permits an already installed device to be reactivated and the agent upgraded to the chosen version.
