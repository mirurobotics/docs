## Upgrade

To upgrade the Miru CLI to the latest version, run the upgrade command in your local machine's terminal.

<Tabs>
  <Tab title="Linux (apt)">
    ```bash
    sudo apt-get update && sudo apt-get install --only-upgrade miru
    ```

    <Info>
      If you originally installed via `curl … install.sh`, you can switch to apt by following the [Linux (apt) install instructions](#install). Both installation methods are kept in sync so switching is safe.
    </Info>
  </Tab>
  <Tab title="Linux (script)">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/mirurobotics/cli/main/install.sh | sh
    ```

    The installation script automatically upgrades to the latest version if the CLI is already installed.
  </Tab>
  <Tab title="macOS">
    ```bash
    brew upgrade mirurobotics/cli/miru
    ```

    <Info>
      If you are not seeing the latest CLI version, run `brew update` first to refresh your local Homebrew tap, then re-run the install or upgrade command.
    </Info>
  </Tab>
</Tabs>

## Verify

To verify the CLI is installed and up to date, run the `miru version` command.

```bash
$ miru version
Version: 0.7.0
Commit: 7ad50107b9af27961e2698b54cc7451d67d3d356
Built: 2025-11-18T04:35:20Z
```

You can find the CLI release changelog in the [CLI changelog](/docs/changelogs/cli).

## Uninstall

<Tabs>
  <Tab title="Linux">
    To remove the CLI but keep the apt source and keyring, run:

    ```bash
    sudo apt remove miru
    ```

    To remove the CLI and purge configuration files, run:

    ```bash
    sudo apt purge miru
    ```

    To also remove the apt source and keyring (clean removal), run:

    ```bash
    sudo rm /etc/apt/sources.list.d/miru-cli.list /usr/share/keyrings/miru-cli.gpg
    sudo apt-get update
    ```

    <Info>
      If you installed via `curl … install.sh` instead of apt, run `sudo rm "$(which miru)"` to remove the binary.
    </Info>
  </Tab>
  <Tab title="macOS">
    ```bash
    brew uninstall miru
    ```
  </Tab>
</Tabs>

Remove any local configuration files.

```bash
rm -rf ~/.miru
```
