## Verify

To verify the CLI is installed and up to date, run the `miru version` command.

```bash
$ miru version
Version: 0.10.0
Commit: 76b320c4ecec7a8e4b12cabb635831a4ad1d5d28
Built: 2026-04-20T02:54:55Z
```

You can find the CLI release changelog in the [CLI changelog](/docs/changelogs/cli).

## Upgrade

To upgrade the Miru CLI to the latest version, run the upgrade command in your local machine's terminal.

<Tabs>
  <Tab title="macOS">
    ```bash
    brew upgrade mirurobotics/cli/miru
    ```

    <Info>
      If you are not seeing the latest CLI version, run `brew update` first to refresh your local Homebrew tap, then re-run the install or upgrade command.
    </Info>
  </Tab>

  <Tab title="Linux">

    #### APT

    Upgrade via `apt`.

    ```bash
    sudo apt-get update && sudo apt-get install --only-upgrade miru-cli
    ```

    <Info>
      The `--only-upgrade` flag ensures the `miru-agent` package isn't unintentionally upgraded.
    </Info>

    #### Manual

    Manual installations require installing the new package manually. The old package is automatically removed.

    <Steps>
      <Step title="Download">
        Download the `deb` package from the [GitHub Releases](https://github.com/mirurobotics/cli/releases) page.
      </Step>
      <Step title="Install">

        ```bash
        sudo dpkg -i <...>.deb
        ```
      </Step>
    </Steps>
  </Tab>

</Tabs>

## Uninstall

<Tabs>
  <Tab title="macOS">
    Remove via Homebrew.

    ```bash
    brew uninstall miru
    ```
  </Tab>

  <Tab title="Linux">
    Remove the `deb` package.

    ```bash
    sudo apt purge miru-cli
    ```

    If installed via `apt`, you can also remove the apt source.

    ```bash
    sudo rm /etc/apt/sources.list.d/miru-cli.sources
    sudo apt-get update
    ```

    <Info>
      The signing key at `/etc/apt/keyrings/miru.gpg` is shared with the Miru Agent. Only remove it if you have also uninstalled the Miru Agent.
    </Info>
  </Tab>
</Tabs>

Finally, remove any local configuration files.

```bash
rm -rf ~/.miru
```
