The Miru Agent is distributed as a Debian package. Other package formats are not yet supported.

We recommend using `apt` to install the agent. However, you can manually install the package by downloading it from GitHub and installing it with `dpkg`.

<Danger>
    [Provisioning tokens](/learn/devices/provision/provisioning-tokens) require Miru Agent `v0.9.0` or later. To install and provision a version that precedes `v0.9.0`, use the legacy [provisioning script](/learn/devices/provision/provisioning-script) instead.
</Danger>
