The CLI is available through Homebrew on macOS and `apt` on Linux. Windows is not supported.

<Tabs>
  <Tab title="macOS">

    Install the CLI via Homebrew.

    ```bash
    brew install mirurobotics/cli/miru
    ```

    <Info>
      If you are not seeing the latest CLI version, refresh Homebrew with `brew update`.
    </Info>
  </Tab>

  <Tab title="Linux">

    The CLI is distributed to Linux as a `deb` package. Other package formats are not yet supported.

    We recommend using `apt` to install the CLI. However, you can manually install the `deb` package by downloading it from GitHub and installing it with `dpkg`.

    #### APT

    <Steps>
      <Step title="Set up">

        Set up Miru's `apt` repository.

        ```bash
        # update the apt repository
        sudo apt-get update
        sudo apt-get install -y ca-certificates curl gnupg
        sudo install -m 0755 -d /etc/apt/keyrings
        curl -fsSL https://packages.mirurobotics.com/apt/miru.gpg \
          | sudo gpg --dearmor -o /etc/apt/keyrings/miru.gpg
        sudo chmod a+r /etc/apt/keyrings/miru.gpg

        # add the repository to your sources list
        sudo tee /etc/apt/sources.list.d/miru-cli.sources > /dev/null <<EOF
        Types: deb
        URIs: https://packages.mirurobotics.com/apt/cli
        Suites: stable
        Components: main
        Architectures: $(dpkg --print-architecture)
        Signed-By: /etc/apt/keyrings/miru.gpg
        EOF
        ```
      </Step>
      <Step title="Install">

        ```bash
        sudo apt-get update && sudo apt-get install miru-cli
        ```
      </Step>
    </Steps>

    #### Manual

    <Steps>
      <Step title="Download">
        Download the `deb` package from the [GitHub Releases](https://github.com/mirurobotics/cli/releases) page.
      </Step>
      <Step title="Install">

        ```bash
        sudo dpkg -i <...>.deb
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
