The CLI is available through Homebrew on macOS and `apt` on Linux. Windows is not supported.

<Tabs>
  <Tab title="macOS">

    Install the CLI via Homebrew.

    ```bash
    brew install mirurobotics/cli/miru
    ```

    <Info>
      If you are not seeing the latest CLI version, refresh Homebrew with `brew update`.
    </Info>
  </Tab>

  <Tab title="Linux">

    The CLI is distributed to Linux as a `deb` package. Other package formats are not yet supported.

    We recommend using `apt` to install the CLI. However, you can manually install the `deb` package by downloading it from GitHub and installing it with `dpkg`.

    #### APT

    <Steps>
      <Step title="Set up">

        Set up Miru's `apt` repository.
        <SetupApt />

        <Info>
          The CLI's `apt` repository configuration is identical to the Miru Agent. If you installed the Miru Agent via `apt`, you can skip this step.
        </Info>

      </Step>
      <Step title="Install">
        ```bash
        sudo apt-get update && sudo apt-get install miru-cli
        ```
      </Step>
    </Steps>

    #### Manual

    <Steps>
      <Step title="Download">
        Download the `deb` package from the [GitHub Releases](https://github.com/mirurobotics/cli/releases/latest) page.
      </Step>
      <Step title="Install">

        ```bash
        sudo dpkg -i <...>.deb
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
