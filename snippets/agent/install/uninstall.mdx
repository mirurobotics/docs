Remove the Debian package.

```bash
sudo apt-get purge miru-agent
```

If installed via `apt`, you can also remove the apt source.

```bash
sudo rm /etc/apt/sources.list.d/miru.sources
sudo apt-get update
```

<Info>
    The signing key at `/etc/apt/keyrings/miru.gpg` is shared with the Miru CLI. Only remove it if you have also uninstalled the Miru CLI.
</Info>
