To verify the installation, run the `miru-agent version` command.

```bash
$ miru-agent version
Version: v0.8.0
Commit: 22282970fc3cf8754da1ecfe885f045809019809
```

Navigating back to the [Devices page](https://app.mirurobotics.com/devices), you should see your device listed with the status `Activating` before transitioning to `Online`.

<Info>
  The transition from `Activating` to `Online` typically takes 5-15 seconds, depending on your network connection.
</Info>
