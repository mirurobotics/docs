To install the Miru CLI, run the installation command in your local machine's terminal.

<Note>
  **apt is the recommended path on Linux.** The install script on the `Linux (script)` tab is provided for containers, Alpine, and CI environments without apt. On a Debian-based host, prefer the `Linux (apt)` tab.
</Note>

<Tabs>
  <Tab title="Linux (apt)">
    Install the prerequisites, add the Miru apt repository, then install the CLI.

    <Info>
      Supported on recent Debian-based distributions.
    </Info>
    {/* TODO: list exact supported distros once the CLI release matrix is published */}

    ```bash
    sudo apt-get install -y ca-certificates curl gnupg
    curl -fsSL https://apt.mirurobotics.com/miru-apt.gpg.pub \
      | sudo gpg --dearmor -o /usr/share/keyrings/miru-cli.gpg
    gpg --show-keys /usr/share/keyrings/miru-cli.gpg
    echo "deb [signed-by=/usr/share/keyrings/miru-cli.gpg] https://apt.mirurobotics.com/cli stable main" \
      | sudo tee /etc/apt/sources.list.d/miru-cli.list
    sudo apt-get update && sudo apt-get install miru
    ```

    <Info>
      {/* TODO: publish fingerprint */}
      `gpg --show-keys` prints the imported key. Confirm the fingerprint matches the published value before proceeding.
    </Info>

    <Info>
      The CLI is signed by the same key as the Miru Agent but installed under a separate keyring (`/usr/share/keyrings/miru-cli.gpg`) so you can manage the two apt sources independently.
    </Info>

    To install or downgrade to a specific version, list available versions and pin the install:

    ```bash
    sudo apt list -a miru
    sudo apt-get install miru=<version>
    ```
  </Tab>
  <Tab title="Linux (script)">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/mirurobotics/cli/main/install.sh | sh
    ```

    <Warning>
      This command pipes remote code into a shell. On systems with apt, prefer the `Linux (apt)` tab, which uses GPG-verified packages.
    </Warning>

    To install a specific version, pass `--version=<semver>`:

    ```bash
    curl -fsSL https://raw.githubusercontent.com/mirurobotics/cli/main/install.sh \
      | sh -s -- --version=0.10.0
    ```

    The script requires `curl`, `tar`, `grep`, and `cut` to be installed. Review the contents of the installation script [here](https://github.com/mirurobotics/cli/blob/main/install.sh).
  </Tab>
  <Tab title="macOS">
    ```bash
    brew install mirurobotics/cli/miru
    ```

    <Info>
      If you are not seeing the latest CLI version, run `brew update` first to refresh your local Homebrew tap, then re-run the install or upgrade command.
    </Info>
  </Tab>
</Tabs>

<Note>
  The Miru CLI supports **macOS** and **Linux**. Windows is not supported.
</Note>
